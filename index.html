Habilidades

HTML

HyperText Markup Language (Linguagem de Marcação de Hipertexto), a linguagem padrão utilizada para criar e estruturar o conteúdo de páginas web, como textos, imagens e links.

CSS

CSS, a sigla para Cascading Style Sheets (Folhas de Estilo em Cascata), é a linguagem que dá vida, cor e forma à internet. Se o HTML é o esqueleto de uma página web, fornecendo a estrutura e o conteúdo, o CSS é a pele, as roupas e a personalidade, definindo toda a estética visual.

JS

JavaScript, ou simplesmente JS, é a linguagem de programação que adiciona interatividade e comportamento às páginas web. Se o HTML é o esqueleto e o CSS é a aparência, o JavaScript é o sistema nervoso e os músculos, permitindo que a página reaja às ações do usuário e se torne dinâmica.

REACT NATIVE

React Native é um framework criado pelo Facebook (agora Meta) que revolucionou o desenvolvimento de aplicativos para celular. Sua principal promessa é permitir que desenvolvedores criem aplicativos para iOS e Android usando uma única base de código, escrita em JavaScript e utilizando a biblioteca React.
